Windows

  • Windows 10 version 1903 or later, Windows 11
  • 64-bit processor (x86-64)
  • 4 GB RAM minimum (8 GB recommended)
  • 100 MB free disk space
  • No internet connection required

Android

  • Android 8.0 (API 26) or later
  • ARM64 processor (arm64-v8a)
  • 2 GB RAM minimum
  • 50 MB free storage

Windows (.exe)

  1. Download VeriForgeRed.exe from the table above
  2. Double-click the downloaded file
  3. If Windows SmartScreen appears, click "More info" then "Run anyway"
  4. Follow the setup wizard
  5. Launch VeriForge Red from the Start Menu or desktop shortcut

Windows (Portable)

  1. Download VeriForgeRed-Portable.zip
  2. Extract the zip file to any folder
  3. Run VeriForgeRed.exe directly from the extracted folder
  4. No installation required. No registry changes.

Android (.apk)

  1. Download VeriForgeRed.apk on your Android device
  2. Open the downloaded file
  3. If prompted, allow installation from unknown sources
  4. Tap "Install"
  5. Open VeriForge Red from your app drawer

To verify the integrity of your download, compare the SHA-256 hash of the downloaded file with the checksum shown in the table above.

Windows (PowerShell)

Get-FileHash .\VeriForgeRed.exe -Algorithm SHA256

Windows (Command Prompt)

certutil -hashfile VeriForgeRed.exe SHA256

Linux / macOS

sha256sum VeriForgeRed.exe

Android (Termux)

sha256sum VeriForgeRed.apk
